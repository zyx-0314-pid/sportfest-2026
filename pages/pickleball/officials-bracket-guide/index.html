How to update pickleball

  1. Open pages/pickleball/assets/js/pickleball-page.js.
  2. Edit names only inside PICKLEBALL_PAIR_ENTRIES. Each pair must contain exactly two names.
  3. Keep every pairId unchanged because progression uses those IDs.
  4. Record a completed match by changing its winningPairId from null to the winning pair ID.

How to change the active courts

Enter exactly two ready match IDs inside ACTIVE_PICKLEBALL_MATCH_IDS. The selected matches must use different courts and both participant pairs must already be resolved.

Important: Enter results in match order before activating matches that depend on those results. Invalid or unresolved progression raises an explicit error.
