Unlisted bracket instructions

How to add or change names

  1. Open pages/badminton/assets/js/badminton-page.js.
  2. Find BADMINTON_PAIR_ENTRIES at the top of the file.
  3. Edit the two values inside the pair's playerNames list. Each pair must always have exactly two names.
  4. Keep every pairId unchanged because the match schedule uses those IDs to move pairs through the bracket.

How to record a result

Find the completed match inside BADMINTON_MATCH_ENTRIES. Change its winningPairId from null to the winning pair ID, such as "pairA". Later matches, winner names, and loser names update automatically.

How to change the active courts

Find ACTIVE_BADMINTON_MATCH_IDS and enter exactly two match IDs—one for each active court. Both matches must already have two resolved pairs. Player avatars use each name as their DiceBear seed and update automatically when a name changes.

Important: Enter results in match order. A later result cannot be resolved until all matches feeding into it have a recorded winner.
